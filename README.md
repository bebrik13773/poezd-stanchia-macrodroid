Поезд станция - это мини приложение которое работает на macrodroid, и позволяет узнавать о следующей станции поезда, пригодится когда едешь на нем. Работает через API [Яндекс расписания.
](https://rasp.yandex.ru/)

#### Функции:
1. Просматривать название, время до прибытия, время стоянки следующих станций
2. Быстро настраивать рейс
3. Автообновляющееся уведомление с нужной информацией
4. Работа без интернета

#### Установка:
1. Установить Macrodroid ([Google Play](https://play.google.com/store/apps/details?id=com.arlosoft.macrodroid))
2. Скачать все файлы из папки ([Яндекс диск](https://disk.yandex.ru/d/87kM5e8x2FZZMQ))
3. Открыть каждый файл через Macrodroid, нажать кнопку + в правом нижнем углу для каждого файла, при добавлении попросит разрешения, необходимо дать все, могут появляться уведомления с просьбой дать разрешения, они необходимы для работы.
4. Установка завершена! Для удобного запуска добавьте ярлык([[#Добавление ярлыка]]) и не забудьте настроить()
#### Добавление ярлыка
1. В приложении Macrodroid перейти на вкладку макросы (*в нижней части экрана*)
2. Долгое нажатие по макросы Поезд станция (Выделен синим цветом)
3. Выбрать: создать ярлык
4. Выбрать иконку (не обязательно)
5. Нажать ок
6. Ярлык добавлен!
#### Настройка:
1. [Перейти в кабинет разработчика Яндекс для создания ключа API](https://developer.tech.yandex.ru/services/4)
2. Авторизоваться через Яндекс ID, если требуется
3. Нажать: новый ключ (*правый верхний угол*)
4. В появшеяся окно вписать любое название, например Поезд станция
5. Скопировать ключ
6. Запустить макрос (см. [[#Добавление ярлыка]])
7. Перейти в настройки
8. Вставить полученный ранее ключ в соответствующее поле
9. Нажать: Выход
10. Ключ добавлен, макрос готов к использованию!
#### Выбор рейса:
1. Запустить макрос (см. [[#Добавление ярлыка]])
2. Перейти в настройки
3. Нажать: Поиск рейсов
4. В первое поле ввести станцию отправления, не ту от которой отправляется поезд, в ту на которой вы садитесь
5. Во второй поле вписать дату отправления в формате: 2026-03-15 (Ту дату когда вы садитесь)
6. Нажать: искать
7. Подождать загрузки (Появится название станции)
8. Используя стрелки выбрать вашу станцию отправления (Точное название указано в билете) из всех найденых, если вместо названия станции что-то типа {lv=ответ entries[1 12]}] то это значит что больше станций не найдено (это баг)
9. Нажать: дальше
10. Ввести название станции прибытия
11. Нажать: искать
12. Подождать загрузки (Появится название станции)
13. Используя стрелки выбрать вашу станцию прибытия (Точное название указано в билете) из всех найденых, если вместо названия станции что-то типа {lv=ответ entries[1 12]}] то это значит что больше станций не найдено (это баг)
14. Появиться загрузка, надо подождать
15. Выбрать ваш рейс из найденых
16. Завершить
17. ДЛЯ ТОГО ЧТОБЫ ДАННЫЕ ОБНОВИЛИСЬ: Главное меню -> Обновить данные -> Подождать -> Данные обновлены!
